ProgramWeb

O ProgramWeb é um site desenvolvido com o framework BootStrap e que serviu como projeto final do curso de Programador Web Básico que fiz em 2020 na Fundação Paulo Feitosa.

O objetivo geral do site é passar informações básicas de HTML, CSS e JavaScript a programadores web iniciantes. E mostrar quais os principais editores de texto mais utilizados pro desenvolvedores.

Nele é possível encontrar um breve resumo da história do HTML, CSS e Javascript. Além disso, ele contém as principais tags HTML, as propriedades CSS, formas de adicionar folhas de estilo a página Web e instruções de programação com JavaScript.

O BootStrap passou por uma atualização, por este motivo algumas das funcionalidades do ProgramWeb precisam ser reescritas e modificadas para que tenham o correto funcionamento.

Histórico Escolar

Esse pequeno programa de Histórico Escolar foi desenvolvido em C++ durante o segundo semestre do curso Técnico em Informática para a disciplina Estrutura de Dados.

O professor passou o código de um sistema mais simples e lançou o desafio de criar um minisistema, onde deveria ter como menu principal as seguintes opções: 1- Inserir; 2 - Atualizar; 3 - Excluir; 4 - Pesquisar; 5 - Listar; 0 - Sair.

Cada opção do menu de 1 a 5 deveria levar a uma nova tela para se trabalhar cada nova finalidade. O minisistema deveria ser todo modular, ou seja, com funções diretas ou auxiliares para cada finalidade.

O minisistema possui uma interface simples, apenas para testar a funcionalidade de cada opção do menu principal e seus submenus.

Clique aqui para acessar o código do programa

Te Controla

O Te Controla é um aplicativo mobile que está em desenvolvimento e irá ser apresentado como PCCT do curso Técnico em Informática.

O principal objetivo do aplicativo é permitir que o usuário faça o controle financeiro pessoal e tenha uma visão das despesas feitas a cada mês.

O aplicativo deve funcionar mais ou menos assim: Usuário faz login no sistema, caso ele não conta no aplicativo, o sistema deve permitir que ele faça o cadastro de uma nova conta. Tendo o feito o login, o usuário pode cadastrar as despesas e escolher a categoria que a despesa melhor se encaixa, os dados dessas despesas irão ser usados para gerar o relatório de todas os gastos feitos pelo usuário.

O aplicativo está sendo desenvolvido com Expo, React-Native (biblioteca Javascript) e utilizará o Firebase como banco de dados.
